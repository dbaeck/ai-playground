AI Playground
=============

Library and demo projects to implement and test various AI related algorithms. Roadmap:

* Search Algorithms
* Learning Algorithms
* tbc.

# Search Algorithms

The Framework should implement the following Algoritms:

* Breadth First Search
* Depth First Search
* Uniform Cost Search
* Iterative Deepening
* Beam Search
* Best First Search
* A*
* Hill Climbing
* Recursive Best First Search
* Real Time A*
* Simulated Annealing
* Local Search
